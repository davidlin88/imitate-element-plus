# @dejavu-ui/components

Dejavu-ui 框架组件模块

## 结构

```
├── components
│   ├── base                    # 基础样式
│   ├── hello-world             # 组件目录
│   │   ├── __tests__           # 测试
│   │   ├── index.ts            # 导出可install的组件
│   │   ├── src                 # 组件代码目录
│   │   │   ├── hello-world.ts  # 组件声明
│   │   │   └── hello-world.vue # SFC
│   │   └── style               # 组件样式
│   │       ├── css.ts          # 组件CSS
│   │       └── index.ts        # 组件Sass
│   ├── index.ts                # 全组件完整导出
│   └── package.json
```
